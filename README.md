# 📊 Data-Driven Stock Analysis

An interactive data analytics project that analyzes **NIFTY-50 stock market performance** using **Python, Streamlit, and Power BI**.
The project focuses on **data cleaning, financial analysis, and visualization**, enabling users to explore market trends, stock volatility, and sector performance through interactive dashboards.

---

# 📌 Project Overview

The goal of this project is to build a **data-driven stock market analysis system** that provides insights into stock performance using historical market data.

The project includes:

* Data extraction and preprocessing
* Financial metric calculations
* Interactive visualizations
* Streamlit dashboard
* Power BI analytics dashboard

This project demonstrates how **data analytics and visualization tools can help investors understand stock trends and make informed decisions**.

---

# 🎯 Objectives

* Analyze **NIFTY-50 stock market data**

* Identify **top performing and underperforming stocks**

# 📊 Key Features

### 📈 Cumulative Return Analysis

Analyzes stock performance over time to determine **overall growth trends**.

### 🔥 Volatility Analysis

Measures the **risk and fluctuation** of stock prices.

### 🏭 Sector-wise Performance

Compares **different sectors in the NIFTY-50 index** to identify strong and weak industries.

### 🔗 Correlation Heatmap

Displays relationships between different stocks to understand **portfolio diversification opportunities**.

### 📅 Monthly Gainers & Losers

Identifies the **best and worst performing stocks** each month.

### 🖥️ Interactive Dashboard

Users can explore stock insights using a **Streamlit web application**.

---

# 🔄 Data Pipeline

The project follows a structured data pipeline:

1. **Raw Data Collection**

   * Stock market data stored in YAML format

2. **Data Conversion**

   * YAML files converted into CSV format

3. **Data Cleaning**

   * Missing values handled
   * Duplicate records removed
   * Structured dataset created

4. **Data Analysis**

   * Calculation of financial metrics
   * Exploratory data analysis

5. **Visualization**

   * Interactive charts using Plotly and Streamlit
   * Business dashboard using Power BI
